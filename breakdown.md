# Confidence Score Breakdown

This document explains exactly how scoring works in Dataverse AI Interview Coach, based on the backend implementation.

## 1. Scoring Layers

There are two levels of scoring:

1. Per-answer scoring (one confidence score per candidate answer)
2. Session-level scoring (aggregate report across all answers)

## 2. Per-Answer Evaluation Inputs

For every answer, the evaluator receives:

1. Current persona context (company style, rewards, penalties)
2. Interview configuration (role, difficulty, interview type, mode)
3. The specific interviewer question
4. The candidate answer
5. Up to 2 retrieved strong-reference examples (RAG) for the same interview type

## 3. 8 Core Dimensions (each scored 0-10)

Each answer is scored on all of the following dimensions:

1. clarity: Clarity
2. structure: Structure (STAR / framework)
3. privacy_terminology: Privacy terminology
4. confidence: Confidence
5. risk_reasoning: Risk-based reasoning
6. regulatory_understanding: Regulatory understanding
7. business_alignment: Business alignment
8. org_context: Org-context awareness

Notes:

1. Dimension scores are integers in range 0-10.
2. The evaluator also returns a short rationale per dimension.

## 4. 6 Elite Principles (hit/miss)

Each answer is also checked against six principles (boolean true/false):

1. enterprise_scale: Enterprise-Scale (Efficiency over Budget)
   Rewards: efficiency, automation, scale; not simple budget cutting.
2. automation: Technological (Proactive Automation)
   Rewards: automated architecture; not manual process or headcount growth.
3. cognitive: Cognitive (Thought process + cross-pollination)
   Rewards: explicit reasoning and transferable thinking.
4. chaos: Chaos (Ambiguity & nimble pivots)
   Rewards: handling uncertainty, pivots, incident judgment.
5. collaboration: Collaboration (Cross-functional diplomacy)
   Rewards: alignment across legal, engineering, product, security.
6. operational_enabler: Operational (Compliance as a business enabler)
   Rewards: framing compliance as trust, velocity, and business outcomes.

The evaluator also returns principle_notes describing misses and why they matter.

## 5. Per-Answer Confidence Score (0-100)

For each answer, the model returns:

1. confidence_score: integer 0-100

Important behavior:

1. This score is generated by the rubric-guided evaluator in the same output that includes dimensions and principles.
2. At difficulty Executive (FAANG bar), principles are hard gates. Missing principles should significantly reduce confidence_score.
3. At lower difficulties, principles are coaching signals and do not hard-cap the score.

## 6. Additional Per-Answer Outputs

Each answer evaluation also returns:

1. stronger_answer: rewritten stronger version of candidate response
2. missed_concepts: specific missing frameworks, regulations, or points
3. star_notes: guidance to improve Situation-Task-Action-Result structure

## 7. Session-Level Aggregation

After multiple answers, the session report is calculated as follows.

### 7.1 Dimension Averages

For each dimension:

dimension_average = round(mean(all answer scores for that dimension), 1)

### 7.2 Principle Hit Rates

For each principle:

principle_hit_rate = round(number_of_answers_with_hit / total_answers, 2)

### 7.3 Overall Confidence

overall_confidence = round(mean(all per-answer confidence_score values))

This is the top-level session readiness score shown as x/100.

### 7.4 Strengths and Weaknesses

1. Strengths are top 3 dimensions with average >= 7.
2. Weaknesses are bottom 3 dimensions with average < 7.

### 7.5 Summary Text

Summary includes:

1. Overall readiness score
2. Strongest dimensions
3. Focus areas
4. Elite principles to develop (principles with hit rate < 0.5)

## 8. Edge Cases

1. If answer text is too short or empty (trimmed length < 5), confidence_score is set to 0 with star_notes indicating no substantive answer.
2. If there are no evaluations yet, session report defaults to zero/empty values.

## 9. Practical Interpretation for Presentations

Use this short explanation:

1. Every answer is scored on 8 dimensions and 6 leadership principles.
2. The system returns a per-answer confidence score from 0-100.
3. Session confidence is the average of those per-answer confidence scores.
4. Executive mode is stricter because principle misses act like hard caps.
